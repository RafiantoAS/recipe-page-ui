# Recipe Page UI

A modern and responsive recipe page built using pure HTML and CSS.

This project presents a complete recipe experience including preparation details, ingredients, step-by-step cooking instructions, and nutrition information within a clean and structured user interface.

The design focuses on content organization, typography hierarchy, spacing, and readability while maintaining a visually appealing layout.
